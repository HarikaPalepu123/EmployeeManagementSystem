# Employee Management System (Java)

## Description
This is a **Java console application** for managing employee data efficiently.  
It demonstrates the use of **Java Collections** and **basic object-oriented programming (OOP) principles**.  

The project allows you to:  
- Store employee details: **ID, Name, Salary**  
- Sort employees using **Comparator** by:
  - Employee ID  
  - Employee Name  
  - Employee Salary  
- Display employee information on the console  

This project is designed for **learning Java Collections**, practicing **OOP concepts**, and preparing for **technical interviews**.

## Technologies Used
- Java  
- Java Collections Framework (ArrayList, Comparator)

## Features
- Add, view, and sort employee details  
- Console-based output  
- Simple and clean structure for easy understanding
